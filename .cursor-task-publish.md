#### What the Plugin Does (Hook by Hook)

Document each hook:

1. **`before_prompt_build`** — Auto-injects vault context + session recap + memory recall mandate + communication protocol. Runs every turn.
2. **`message_sending`** — Filters outbound messages (removes banned phrases like "good catch", "great question"). Auto-rewrites questions when memory already has the answer.
3. **`gateway_start`** — Runs startup recovery. Detects context death and surfaces recovery notice.
4. **`session_start`** — Fetches session recap for context injection. Triggers weekly reflection if due.
5. **`session_end`** — Clears session state.
6. **`before_reset`** — Auto-checkpoints before `/new`. Runs observer if enabled. Extracts facts.
7. **`before_compaction`** — Runs observation before context compaction.
8. **`agent_end`** — Runs observation after heartbeat agent turns.

#### Registered Tools

The plugin registers two tools:
- **`memory_search`** — Search vault with `query`, `maxResults`, `minScore`, `sessionKey`
- **`memory_get`** — Read vault files with `path`/`relPath`, `from`, `lines`
